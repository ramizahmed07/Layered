# Layered

An interactive MDX blog built with Next.js and React Server Components.

## What's cool about it?

- Write posts in MDX and embed custom React components
- Build interactive blog content, not just static text
- Super fast — 100 Lighthouse score & all green on PageSpeed
- Flicker-free dark mode
- Deploys to Vercel in ~30s

## Tech

- Next.js (App Router)
- React.js
- MDX
- Vercel

## Idea

Most blogs are static.
Layered lets you build interactive experiences inside your posts.
